APP DEVELOPMENT

JAVA

Java is the language in which complete android dev is done.

Now, Kotlin a programming language by Google, is also introduced which is used in place of JAVA.

XML

In XML you write code which is mainly used for designing your app interface known as UI (User interface )

Android SDK

Basically it is a java module that gives developers access to device function like camera and accelerometer.

It also contains library called Gradle. (Search more about this on Google )

Android studio

It is the IDE ( integrated development kit) . It is place where you build, tests, runs etc your android app.

Means on Android studio you build your apps that will finally be available for common people for use.

Database

Cloud services like Firebase provides simple API's to store data in cloud and make it available on devices.

Some more thing you should know for Android development are

1)Fragments, Recycler view, Adaptors

2)Working with multi screen apps

3)Material design

4)Android activity and it's life cycle

5)FCM or GCM for push notifications
